# Visual Review 视觉验收规范

## 设计哲学

技术验证（schema/overflow/formula）只是基础。
视觉验收回答一个核心问题：**这个作品真的好看吗？**

Agent在交付前必须进行视觉自检，不通过则修改。

---

## 通用视觉验收清单（所有格式）

### 1. 默认模板感检测

| 检查项 | 通过标准 | 失败时的修改 |
|--------|----------|-------------|
| 配色是否AI典型 | 无紫色渐变/白紫/纯蓝#0000FF | 换用预设palette |
| 字体是否默认 | 无Calibri/Arial默认感（除数据表格外） | 使用设计档案字体 |
| 布局是否单调 | 无满屏文字/无层次 | 增加留白、改变字号对比 |
| 封面是否平庸 | 非纯文本/非纯白背景 | 增加视觉元素 |

### 2. 拥挤度检测

| 检查项 | 通过标准 | 失败时的修改 |
|--------|----------|-------------|
| 页边距充足 | 四周≥5%页面宽度 | 增大页边距 |
| 行距舒适 | 正文字高的1.4-1.8倍 | 调整lineHeight |
| 段落间距清晰 | 段落间有明显间隔 | 增加段落间距 |
| 元素不重叠 | 无文字被遮挡 | 调整布局 |
| 信息密度合理 | 不高到窒息/不低到空洞 | 调整内容量或分页 |

### 3. 对比度检测

| 检查项 | 通过标准 | 失败时的修改 |
|--------|----------|-------------|
| 文字vs背景 | 对比度≥4.5:1（正文），≥3:1（大标题） | 调整颜色 |
| 标题vs正文 | 标题明显更重/更大 | 增大字号差或加粗 |
| 强调色醒目 | 强调色与主色有足够区分 | 调整饱和度或色相 |
| 无低对比陷阱 | 浅灰文字不用在浅灰背景上 | 调深文字或调浅背景 |

### 4. 专业感检测

| 检查项 | 通过标准 | 失败时的修改 |
|--------|----------|-------------|
| 对齐精准 | 所有元素左/右/中对齐一致 | 使用网格对齐 |
| 间距一致 | 同类元素间距相等 | 统一间距值 |
| 无视觉噪音 | 无多余线条/无意义装饰 | 删除多余元素 |
| 图表有解释力 | 图表不空洞、数据有意义 | 补充数据或调整图表类型 |
| 封面有第一眼吸引力 | 3秒内能抓住注意力 | 增强视觉冲击 |

## 格式特定视觉验收

### DOCX 视觉验收

| # | 检查项 | 权重 |
|---|--------|------|
| 1 | 封面有独特背景（渐变/几何/模糊） | 必须 |
| 2 | 封面标题与背景对比度足够 | 必须 |
| 3 | 三级颜色层级清晰（标题/正文/注释） | 必须 |
| 4 | 表格不使用Excel默认蓝色表头 | 必须 |
| 5 | 三线表顶线粗、底线粗、表头下线细 | 必须 |
| 6 | 页眉页脚完整、页码格式正确 | 必须 |
| 7 | 图片尺寸合适（不变形、不过小） | 必须 |
| 8 | 段落首行缩进2字符（中文） | 建议 |
| 9 | 封面标题字间距充足（不拥挤） | 建议 |
| 10 | 表格交替行颜色增强可读性 | 建议 |

### XLSX 视觉验收

| # | 检查项 | 权重 |
|---|--------|------|
| 1 | 网格线已隐藏 | 必须 |
| 2 | 内容从B2开始（四周留边） | 必须 |
| 3 | 表头使用深色背景+白字 | 必须 |
| 4 | 交替行颜色（白/#F9F9F9） | 必须 |
| 5 | 图表有标题和轴标签 | 必须 |
| 6 | 条件格式增强关键数据 | 建议 |
| 7 | 封面页有报告标题和关键指标 | 建议 |
| 8 | 图表系列色与整体配色协调 | 建议 |

### PDF 视觉验收

| # | 检查项 | 权重 |
|---|--------|------|
| 1 | 封面占满整页（无白边） | 必须 |
| 2 | 正文11pt、行高1.7 | 必须 |
| 3 | 三线表规范（顶粗底粗头细） | 必须 |
| 4 | 页眉页脚完整 | 必须 |
| 5 | 无CSS Counter导致的编号异常 | 必须 |
| 6 | 图表≤8节点或已分页 | 必须 |
| 7 | 段落无孤行寡行 | 建议 |
| 8 | 封面风格与文档类型匹配 | 建议 |

### PPTX 视觉验收

| # | 检查项 | 权重 |
|---|--------|------|
| 1 | 正文≥18px | 必须 |
| 2 | 标题与正文字号差异显著 | 必须 |
| 3 | 一页一个核心信息 | 必须 |
| 4 | 图表优先于文字要点 | 必须 |
| 5 | 无文字溢出/遮挡/漂移 | 必须 |
| 6 | 配色不AI典型（无紫/渐变紫蓝） | 必须 |
| 7 | 封面有"第一眼震撼" | 必须 |
| 8 | 每页3-6个高亮数据点 | 建议 |
| 9 | 正文≤40-50汉字 | 建议 |
| 10 | 目录页非干瘪列表 | 建议 |

---

## 视觉验收执行流程

```
技术验证通过
  → 视觉自检（本清单）
    → ALL "必须"项通过 → 进入交付
    → ANY "必须"项失败 → 修改后重检
    → "建议"项<50%通过 → 记录改进点
```

## AI典型问题快速修复指南

| 问题 | 症状 | 快速修复 |
|------|------|----------|
| 紫色塑料味 | 主色是紫/渐变紫 | 换成corporate/ocean/earth palette |
| 默认模板感 | 看起来像未修改的模板 | 换封面风格、改配色、调字号 |
| 拥挤窒息 | 元素太多、边距太小 | 增大页边距20%、减少每页内容 |
| 层次不清 | 标题正文看起来差不多 | 增大字号差（标题36→正文21） |
| 图表空洞 | 图表无数据或数据太少 | 补充真实数据、增加数据点 |
| 对齐灾难 | 元素参差不齐 | 使用网格布局、统一间距值 |
| 对比度低 | 文字看不清 | 调深文字或调浅背景 |
| 无品牌感 | 看起来像通用文档 | 使用特定palette、增加logo占位 |
