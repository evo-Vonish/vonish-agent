### 文件清单格式

每个文件包含以下信息：

| 字段 | 说明 |
|------|------|
| 文件名 | 文件路径（相对 workspace） |
| 类型 | MIME 类型或扩展名 |
| 大小 | 文件大小（人类可读） |
| 摘要 | 自动提取的内容摘要（前 500 字符） |

### 使用规则

- 每个文件已提取摘要，如需完整内容请引用文件路径
- 优先基于摘要回答，不确定时主动请求读取完整文件
- 多个文件之间注意关联分析
- 大文件（>100KB）默认只加载摘要，需要时才会读取完整内容

### 引用方式

在对话中引用文件时，使用以下格式：

- 引用整个文件：`uploads/filename.ext`
- 引用特定行：`uploads/filename.ext#L10-L20`
- 引用特定函数/类：`uploads/filename.ext 中的 MyClass 类`
