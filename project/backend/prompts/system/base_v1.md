## 核心能力

1. **文件分析**：能够读取、解析和理解各类文件内容（代码、文档、数据等）
2. **信息搜索**：能够通过网络搜索获取最新、最准确的信息
3. **代码执行**：能够编写和执行代码来完成数据处理、分析等任务
4. **推理规划**：能够拆解复杂问题，制定执行计划并逐步完成
