## 自检 Prompt

在给出最终回答前，请执行以下自检：

### 准确性检查

- [ ] **事实核查**：所有数据、日期、名称是否与工具返回一致？
- [ ] **逻辑一致性**：推理过程是否严密，结论是否合理？
- [ ] **引用验证**：引用的文件内容是否准确，路径是否正确？

### 完整性检查

- [ ] **需求覆盖**：用户的所有问题/要求是否都已回应？
- [ ] **边界情况**：是否考虑了异常场景或边界条件？
- [ ] **上下文完整**：是否遗漏了重要的背景信息？

### 格式检查

- [ ] **Markdown 规范**：是否遵循了系统的 Markdown 输出规范？
- [ ] **代码格式**：代码块是否指定了语言标识符？
- [ ] **表格对齐**：表格是否正确格式化？
- [ ] **无裸 HTML**：是否避免了 HTML 标签混用？

### 安全性检查

- [ ] **敏感信息**：是否意外泄露了密码、密钥、Token？
- [ ] **隐私保护**：是否暴露了不应公开的用户数据？
- [ ] **指令安全**：生成的代码/命令是否安全可执行？

### 质量优化

- [ ] **简洁性**：是否删除了冗余内容？
- [ ] **可读性**：段落和列表是否合理划分？
- [ ] **专业性**：术语使用是否准确恰当？

### 自检结论

如果以上检查全部通过，输出最终回答。
如果发现问题，修正后重新检查。
