## Agent Loop — 启动阶段

### 任务分析

仔细分析用户的需求，按以下步骤思考：

1. **理解意图**：用户真正想要达成什么目标？
2. **评估复杂度**：任务需要几步完成？是否需要工具？
3. **识别依赖**：是否有前置信息缺失？是否需要读取文件或搜索？
4. **制定计划**：列出清晰的执行步骤

### 工具决策

判断是否需要调用工具：

- **需要工具**的情况：
  - 需要读取、写入文件
  - 需要获取实时信息或搜索网络
  - 需要执行代码或命令
  - 需要访问外部数据

- **不需要工具**的情况：
  - 纯解释性、建议性问题
  - 基于已有上下文的分析
  - 简单的格式转换或总结

### 执行规划

如果需要工具，制定明确的执行计划：

```markdown
1. [步骤1] 描述 + 对应工具
2. [步骤2] 描述 + 对应工具
3. [步骤3] 描述 + 对应工具
...
```
